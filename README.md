# PalindromZahlen

Suche Palindrom-Zahlen.

## Rechenregel

Sucht zu einer Zahl ihre Palindrom-Zahl
(Zahlen, die von vorne wie von hinten gleich sind, beispielsweise 1221, 1234321, ...),
indem zur Zahl ihre Spiegelzahl addiert wird.

        Beispiel:

         12 -->  12 +  21 =  33 ✅️

         83 -->  82 +  28 = 110 🔄
        110 --> 110 +  11 = 121 ✅️
